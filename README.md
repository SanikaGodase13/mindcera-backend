# MindCera Backend

MindCera is an AI-powered mental wellness platform developed for the MIT India Hackathon.

## Features Implemented

* User Registration
* User Login
* JWT Authentication
* User Profile API
* Forgot Password
* OTP Email Verification
* Password Reset
* PostgreSQL Database Integration
* FastAPI Backend

## Tech Stack

* FastAPI
* PostgreSQL
* SQLAlchemy
* JWT Authentication
* Gmail SMTP
* Python
